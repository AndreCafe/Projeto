Backup - Cópia de Dados

 Topo  Voltar  Avançar

É imprescindível realizar diariamente uma cópia dos arquivos do banco de dados do NexCafé. A qualquer momento seu computador pode sofrer algum tipo de pane de hardware ou software e ter seus dados danificados.

 

O NexServ possui uma opção de realizar a cópia dos dados sem ter que interromper o uso do sistema e sem atrapalhar o atendimento dos clientes. O sistema realiza a cópia para um local destino que você informar. O local padrão é a pasta COPIA que fica dentro da pasta NEXCAFE do seu servidor. Ao realizar o backup, o sistema cria uma pasta cujo nome é a composto pela data e hora do momento de inicio da cópia e coloca dentro dessa pasta os arquivos de dados do NexCafé. É de sua responsabilidade realizar essa cópia em um local seguro. Por exemplo, de nada adianta ter uma cópia dentro do mesmo computador pois ele pode pifar a qualquer momento. O mais seguro é copiar para outro computador ou para um CD.
