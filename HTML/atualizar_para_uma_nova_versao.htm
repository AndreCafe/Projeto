Como atualizar para uma nova versão ?

 Topo  Voltar  Avançar
1.Baixe do site os instaladores do servidor e do administrador. Não é necessário baixar o NexGuard pois as máquinas se atualizam automaticamente depois que é instalado a nova versão do servidor.
2.Se você estiver usando o NexAdmin é necessário fecha-lo antes de instalar a versão nova.
3.Encerre o servidor do NexCafé.
4.Execute o instalador do NexServ e depois do NexAdmin.
5.Pronto !

 

Dúvidas comuns sobre a atualização:

▪Posso atualizar para uma nova versão do NexCafé com a loja em funcionamento ?
▪É necessário desinstalar o NexCafé antes de atualizar para uma nova versão ?
▪Preciso fazer uma cópia dos dados antes de atualizar para uma nova versão ?
▪Se eu instalar uma nova versão do NexCafé vou perder meus dados ?
