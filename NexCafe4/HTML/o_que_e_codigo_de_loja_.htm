O que é código de loja ?

 Topo  Voltar  Avançar

É um código que identifica cada loja usuária do NexCafé. Desde o momento que você faz o registro gratuito do NexCafé você ganha esse código. Ele código continua igual mesmo após a compra definitiva do sistema.
