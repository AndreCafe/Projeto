Quais são os pré-requisitos para instalar o NexCafé ?

 Topo  Voltar  Avançar

Confira as configurações necessárias para usar o NexCafé nos seus computadores:

 

1- Servidor: NexServ

 

Mínimo Necessário:

 

•Sistema Operacional: Windows 98, Me, XP, 2000, 2003, Vista
•CPU: Pentium III 500 Mhz
•Memória (RAM): 128 MB
•Disco Rígido: 300MB espaço livre

Recomendado:

 

•Sistema Operacional: Windows XP ou Vista.
•CPU: acompanha as recomendações do seu sistema operacional
•Memória (RAM): 512 MB
•Disco Rígido: 2gb

 

2- Administrador: NexAdmin

 

Mínimo Necessário:

 

•Sistema Operacional: 2000, XP, 2003, Vista
•CPU: Pentium III 500 Mhz
•Memória (RAM): 128 MB
•Disco Rígido: 20MB espaço livre
•Tela: 800x600

 

Recomendado:

 

•Sistema Operacional: Windows XP ou Vista.
•CPU: acompanha as recomendações do seu sistema operacional
•Memória (RAM): 512 MB
•Disco Rígido: 2gb
•Tela: 1024x600

 

3- Cliente: NexGuard

 

Mínimo Necessário:

 

•Sistema Operacional: Windows 98, Me, XP, 2000, 2003, Vista
•CPU: Pentium II 300 Mhz
•Memória (RAM): 128 MB
•Disco Rígido: 20MB espaço livre

 

Recomendado:

 

•Sistema Operacional: Windows XP.
•CPU: acompanha as recomendações do seu sistema operacional
•Memória (RAM): 256 MB
•Disco Rígido: 2gb
