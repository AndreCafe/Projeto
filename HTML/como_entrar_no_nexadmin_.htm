Como abrir o NexAdmin ?

 Topo  Voltar  Avançar
1.Utilize o ícone do NexAdmin da área de trabalho do Windows ou via Menu Iniciar -> Programas -> NexCafé.

2.Na tela de abertura, informe seu nome de usuário e senha e clique no botão "Login". O usuário padrão é "admin" com a senha em branco.

 

Se tiver problemas para fazer o login veja o item "Porque o NexAdmin não consegue se conectar ao servidor ?"
