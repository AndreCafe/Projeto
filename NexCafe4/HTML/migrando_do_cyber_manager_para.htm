Como migrar do Cyber Manager para o NexCafé

 Topo  Voltar  Avançar

A mudança se resume basicamente em instalar o NexCafé, importar o banco de dados do Cyber Manager e digitar as informações de registro da loja (código e senha) no NexCafé.

 

Pré-requisitos

▪É necessário saber seu código de loja e senha de registro. Se não lembra dessas informações entre em contato com a Nextar antes de iniciar a mudança de programa.
▪Durante a atualização o servidor do Cyber Manager deve estar ativado.

 

Dúvidas comuns sobre a mudança de programa
▪Porque mudar para o NexCafé?
1.O NexCafé tem vários recursos novos e ficou muito mais fácil atender seus clientes. Clique aqui e veja a lista completa de novidades.
2.O Cyber Manager não vai mais evoluir: correções de erros e novos recursos serão implementados apenas no NexCafé.
3.A partir de 01/10/2007 o serviço de suporte técnico será oferecido apenas para o NexCafé.
4.A atualização é grátis!
▪Quero conhecer o NexCafé para me familiarizar com as diferenças antes de mudar de programa, como posso fazer isso?

Simplesmente instale e teste. O NexCafé não interfere no funcionamento do Cyber Manager. Instale o servidor e o administrador do NexCafé na mesma máquina onde você já tem o Cyber Manager e depois de se acostumar com ele realize a mudança de programa. Apenas nas máquinas clientes é que você não poderá estar com os 2 programas simultaneamente.

▪Preciso pagar para fazer a atualização ?

Não. A atualização para a versão 2.0 do NexCafé é grátis para todos clientes registrados do Cyber Manager.

▪Vou perder meus dados se atualizar para o NexCafé?

Não. Todos dados armazenados no Cyber Manager são transferidos para o NexCafé.

▪Devo instalar na mesma pasta do Cyber Manager?

Não. O NexCafé deve ser instalado em sua própria pasta.

▪Posso instalar e simplesmente copiar o banco de dados do Cyber Manager dentro do NexCafé?

Não. Os bancos dados não são compatíveis. Você deve fazer a importação conforme descrito mais abaixo.

▪Posso atualizar para o NexCafé com a loja em funcionamento?

Não é recomendado pois é necessário desinstalar o CMGuard de cada máquina cliente para instalar o NexGuard e isso poderia incomodar seus clientes.

▪Preciso desinstalar o Cyber Manager antes de instalar o NexCafé?

É necessário estar com o Cyber Manager instalado e ativado para poder realizar a mudança no servidor. Mas nas máquinas clientes você precisa remover o CM-Guard antes de instalar o NexGuard.

 

Aprenda a mudar de programa passo a passo

1.Instale o servidor e o administrador do NexCafé no mesmo computador onde está instalado o Cyber Manager.
2.Faça a importação do banco de dados do Cyber Manager.
3.Desinstale o Cyber Manager CM-Guard de cada máquina cliente.
4.Transfira o registro do Cyber Manager para o NexCafé.
5.Instale o NexCafé NexGuard em cada máquina cliente.
6.Pronto!
