Como desinstalar o Cyber Manager CM-Guard das máquinas clientes ?

 Topo  Voltar  Avançar
1.Entre no administrador do Cyber Manager e clique no botão Desligar

 

 

2.Selecione a operação "1. Fechar CM-Guard"
3.Selecione "Todas as Máquinas"
4.Clique em OK

 

 

Vá em cada máquina cliente e faça o seguinte:

 

5.Entre no Menu Iniciar do Windows, depois em Programas, Cyber Manager e clique na opção "Desinstalar Cyber Manager CM-Guard".
6.Responda sim na confirmação.
7.Reinicie a máquina
8.Pronto!
